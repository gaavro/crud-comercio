# Uso
- Colar o código em `app.py`

- Iniciar e entrar no venv

```bash
python -m venv venv
source venv/bin/activate
```

- **NÃO** se esqueça de instalar o flask

- Rodar o **pytest**

```bash
pytest -svv
```

-   **NÃO** se esqueça de criar o requirements.txt
-   **NÃO** se esqueça de criar o .gitignore
